# Forex Trade Tracker

A full-stack web application for tracking forex trades, managing trade history, and analysing trading performance.

This project is designed to help traders log trades, review strategies, and improve decision-making through structured data tracking.

---

## 🚀 Overview

The application consists of two main parts:

- **Client:** React frontend (UI & user interaction)
- **Server:** Node.js + Express backend (API, authentication, database)

## 🛠️ Tech Stack

### Frontend

- React (JavaScript)
- Custom CSS (modular structure)
- React Hooks

### Backend

- Node.js
- Express
- MongoDB
- Mongoose
- JWT Authentication
- bcrypt (password hashing)

---

## 🔐 Core Features

### Authentication

- User registration
- User login
- JWT-based session handling

### Trade Management

- Create trades
- View all trades
- Edit trades
- Delete trades

### UI / UX

- Dashboard-style layout
- Clean and minimal design
- Profit/Loss visual indicators:
  - 🟢 Profit
  - 🔴 Loss
  - ⚪ Breakeven

## 📊 Use Case

This application allows traders to:

- Analyse strategies
